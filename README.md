# Karin Plugin Lux

Karin Plugin Lux 是基于 [KarinJS](https://github.com/KarinJS/Karin) 的插件集合，为机器人提供轻量而实用的扩展能力。

## 安装

```bash
pnpm add karin-plugin-lux -w
```
